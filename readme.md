# Webpack template

## Features

- Basic files support (HTML, CSS, Stylus, JS, Images, Fonts)
- Extracted CSS
- Transpilled JS
- Sourcemaps
- Clean dist
- Dev server (with local network support)
- Static Files
- Cache Breaker

## Instructions

Install dependencies

```npm i```

Launch dev server

```npm run dev```

Build for production

```npm run build```
